# Payment Processor
Сервис для асинхронной обработки платежей. Сервис принимает запросы на оплату, обрабатывает их через внешний платежный шлюз (эмуляцию) и уведомляет клиента о результате через webhook.

## Запуск

1. Заполнить файл `.env` по примеру из `.env.template` в директории `src/`.
2. Поднять все сервисы через `docker-compose.yml`:
    ```
    docker compose up -d
    ```
3. После запуска с портами по умолчанию спецификация к API сервиса Payment Processor будет доступна по адресу http://127.0.0.1:8000/docs
